UX Strategy for iPad Product Design
What I think!!!

UX Process will not be same for every application, specifically ENTERPRISE APPS, its enough to focus on right process areas instead of focusing on every process areas.

Created UX for eliminating Manual Scavenging through Technology and believe NO Human should work in HUMAN SHIT

User Research

  • Supervisor - Minimum Systems Knowledge
  • Fieldworkers - NO Systems Knowledge

User Persona

  • Supervisors - Have Bachelors / Masters, Old, OK to learn few things
  • Field Workers - Not Educated, Not willing to learn

Content Strategy

  • Dont give too many details or workflows.
  • Give only what is required to complete tasks.

Information Architecture

  • Supervisors - Login --> Filter Jobs --> View Job Details
  • Supervisors - Login --> Dashboard --> Search/Filter/Select Contents
  • Supervisors - Login --> Incident Timelines --> Search/Filter/Select Contents
  • Supervisors - Login --> FieldWorkers Status --> Search/Filter/Select Contents
  • Supervisors - Login --> Filter Jobs --> View Job Details
  • Field Workers - Login --> View Jobs --> Select Job --> Choose Vehicle --> Collect Smartballs --> Complete Jobs
  • Field Workers - Login --> My Jobs-> My Job Analytics

Tools Used

  • Adobe XD

Visual Design

  • Color Schema - Chose based on familiarity and environment app will be operated
  • Typography - SanFrancisco
  • UI Components - Simple UI Components

Animations

  • Applied only at right places to get USERS ATTENTION

Design Considerations

  • Human Language, Emotions Mapping
  • No more than 2 layers of Navigation
  • Map Real World with Machine World
  • Not Too many Details - Only What is required to experience on-the-go with 10 inch screen
  • Motivational quotes in landing screen

Testing, Learning and outcomes

Test Strategy

  • Test - Only with People who are not familiar with computer systems
  • Challenges Faced - Make Testers understand how the system works, Know your role and how efficiently you completed given tasks
  • Test Outcome - Human Emotion Language to be mapped with Machine Communication Language
  • Test Outcome - Sometimes unfamiliar with buttons or UI controls used

Learning and Outcomes

  • Tool level Learning - XD, Photoshop, Illustrator
  • Use 1 or 2 Auth Mechanisms - either Fingerprint or Face ID or PIN
  • Color Support during Bright Sunlight and Dark Evenings
  • Typography for Good Readability
  • More Minimalism with right content strategy
